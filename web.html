How Web Works Exercise

Part One: Solify Terminology

  1. HTTP is hyptertext transfer protocol; how to fetch resourcs like html docs
  2. URL is uniform resource locator - the address
  3. DNS is the domain name system
  4. A query string sets the parameters of the link
  5. Get and Post
    • Get - requests that don't change server data
    • Post - requests that change server data
      • sending mail
      • charging credit card
  6. HTTP request is a message from a client sent to initiate an action on the server
  7. HTTP response is a response from the server to the client; GIVE THE CLIENT WHAT THEY WANT
  8. HTTP header let the client/server add parameters into the request/response
    • request- host, cookie
    • response: set-cookie, last-modified
  9. When you type in a website, it'll request the sources
    1. HTML
    2. CSS
    3. JS
